# CodeAlpha Frontend Development Internship Tasks

This repository contains the projects completed as part of my Frontend Development Internship at CodeAlpha.

A responsive personal portfolio website showcasing my skills, projects, achievements, and contact information.

- About and Skills sections

- Contact information
- Smooth navigation

### Technologies Used

- CSS3
- JavaScript

## Task 2: Responsive Image Gallery

An interactive and responsive image gallery with category-based filtering and a lightbox preview.

### Features

- Responsive image grid
- Category filters: All, Nature, Travel, and City
- Full-screen lightbox preview
- Previous and Next image navigation
- Close button
- Keyboard navigation support
- Smooth hover effects and transitions

### Technologies Used

- HTML5
- CSS3
- JavaScript
